# .learnings - 经验教训库

## 目录结构

| 目录 | 内容 | 管理元技能 |
|------|------|-----------|
| errors/ | 错误记录 | evolution-engineer |
| corrections/ | 施欢纠正记录 | evolution-engineer |
| best-practices/ | 最佳实践 | quality-auditor |

## 记录规则

### errors/
- 格式：`YYYY-MM-DD.md`
- 内容：错误描述、原因分析、教训

### corrections/
- 格式：`YYYY-MM-DD.md`
- 内容：施欢原话、纠正原因、修正措施

### best-practices/
- 格式：`YYYY-MM-DD.md`
- 内容：成功案例、可复用模式、最佳方法

## 触发条件

- 任何错误发生 → 记录到errors/
- 施欢纠正 → 记录到corrections/
- 成功完成复杂任务 → 记录到best-practices/
