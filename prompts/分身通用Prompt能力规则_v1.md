# 分身通用 Prompt 能力规则 v1

## 目标
把欢喜的自然语言需求，翻译成模型最容易做对的任务表达。

## 通用翻译结构
1. 任务目标：这次到底要完成什么
2. 输出格式：最终要交付什么形式
3. 风格要求：语气/审美/调性/品牌感
4. 关键约束：必须有、不能有、优先级
5. 交付标准：什么才算完成

## 通用模板
```text
任务目标：
请完成【具体任务】。

输出格式：
请输出为【文案/表格/图片方案/飞书文档/结构化清单】。

风格要求：
整体风格需满足【品牌调性 / 用户偏好 / 场景语气】。

关键约束：
- 必须包含：
- 禁止出现：
- 优先处理：

交付标准：
满足以下条件才算完成：
1.
2.
3.
```

## 分身专项适配
- 内容官：强化标题、结构、情绪张力、转化目标
- 灵感官：强化方向发散、关键词扩展、灵感源头
- 设计官：强化构图、光影、色彩、留白、负面约束
- 系统官：强化准确性、步骤、风险点、验证方式

## 太阳职责
若欢喜原话不够适合模型直接执行，由太阳先翻译，再派发。
