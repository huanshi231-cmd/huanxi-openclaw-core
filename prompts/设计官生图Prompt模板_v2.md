# 设计官生图 Prompt 模板 v2

## 核心目标
把欢喜的人话需求，自动翻译成 Gemini / 其他生图模型最容易做对的提示词。

## 固定生成结构
```text
任务目标：
请生成一张用于【使用场景】的【图片类型】。

主题核心：
- 主题：
- 核心情绪：
- 想传达的感受：

主体设定：
- 主体是谁：
- 年龄/气质：
- 动作状态：
- 面部/姿态：

场景设定：
- 场景地点：
- 时间：
- 环境元素：

构图设定：
- 比例：16:9 / 4:3 / 1:1
- 景别：远景 / 中景 / 近景
- 主体位置：偏左 / 偏右 / 居中
- 留白位置：上方 / 下方 / 左右
- 画面密度：简洁 / 适中 / 丰富

视觉设定：
- 风格：写实电影感 / 治愈 / 高级 / 极简
- 光影：柔光 / 逆光 / 窗边光 / 黄昏光 / 阴天漫反射
- 色彩：主色 / 辅色 / 避免色
- 质感：细腻 / 干净 / 高清 / 有空气感

交付约束：
- 无字底图
- 无水印
- 无logo
- 无畸形手
- 无多余人物
- 无杂乱背景
- 不要过满构图
- 适合后期中文排版

输出要求：
- 高清
- 适配尺寸：1280×720（默认公众号封面）
- 保证上方和下方留白足够
```

## 欢喜语言自动翻译词典
- 高级感 → 干净、克制、留白、统一色系、不过度装饰
- 治愈感 → 柔和光线、细腻情绪、低刺激色彩、安静氛围
- 不要太满 → 主体不过大、背景简洁、边缘留空间
- 有呼吸感 → 留白明显、画面疏密有节奏、不过度堆元素
- 有力量但不压迫 → 稳定构图、柔中带韧、光影有支撑感

## 公众号封面专用加强版
```text
请生成一张公众号封面无字底图。
主题：【主题】
核心情绪：【情绪】
主体：【主体】
场景：【场景】
构图：16:9 横版，主体偏左或偏右，画面上方和下方保留明显留白，适合后期排中文标题和副标题。
风格：写实电影感，治愈，高级，细腻，安静，有情绪张力但不压迫。
光影：【光影】
色彩：主色【主色】，辅色【辅色】，避免荧光色和脏灰色。
要求：无文字、无水印、无logo、无畸变、无多余元素、高清、适合公众号封面使用。
```

## 自动质检清单
- 是否把“人话需求”翻译成可执行画面？
- 是否明确了主体、场景、情绪？
- 是否明确了留白和构图？
- 是否补上了负面约束？
- 是否适合后期加中文？
