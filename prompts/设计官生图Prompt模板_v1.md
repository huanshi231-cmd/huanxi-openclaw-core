# 设计官生图 Prompt 模板 v1

## 用途
用于公众号封面图、文章配图、小红书封面等视觉生成任务。

## 固定模板
```text
请为【任务主题】生成一张【图片类型】。

主体设定：
- 主体：
- 性别/年龄感：
- 动作/状态：
- 情绪关键词：

场景设定：
- 场景地点：
- 时间氛围：
- 关键环境元素：

构图设定：
- 画幅比例：
- 镜头景别：
- 主体位置：
- 留白位置：
- 层次关系：前景 / 中景 / 背景

视觉风格：
- 风格：写实 / 电影感 / 极简 / 治愈 / 高级感
- 光影：柔光 / 逆光 / 窗边光 / 黄昏光 / 阴天漫反射
- 色彩系统：主色 / 辅色 / 避免色
- 材质质感：

交付要求：
- 无字底图
- 不要水印
- 不要logo
- 不要畸形手
- 不要多余人物
- 不要杂乱背景
- 高清
- 适配尺寸：1280×720 或按任务指定

如需适配公众号封面，请确保：
- 上方有标题留白
- 下方有副标题/署名留白
- 画面中心稳定，不要过满
```

## 公众号封面专用简版
```text
请生成一张公众号封面无字底图。
主题：【主题】
主体：【主体】
情绪：【情绪】
场景：【场景】
构图：16:9 横版，主体偏左/偏右，上下保留大面积留白，适合后期排中文标题。
风格：写实电影感，干净，高级，情绪细腻。
光影：【光影】
色彩：主色【主色】，辅色【辅色】，整体统一，不花。
要求：无文字、无水印、无logo、无畸变、无多余元素、高清。
```

## 输出前自检
- 是否明确主体？
- 是否明确情绪？
- 是否明确场景？
- 是否明确留白？
- 是否明确尺寸？
- 是否明确负面约束？
- 是否适合后期排中文？
