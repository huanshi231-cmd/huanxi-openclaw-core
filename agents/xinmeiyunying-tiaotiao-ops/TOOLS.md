## 可用工具原则
- 优先使用当前工作区已有脚本、知识库、模板。
- 外部 API / 发布 / 删除 / 权限调整必须谨慎，必要时交太阳确认。
- 文件改动前先备份；重要结果写入 daily_log 或 memory。

## 本角色重点工具方向
- 全平台发布排期与执行
- 账号资料与栏目管理
- 内容数据盘查与复盘
- 爆款/低效内容诊断
- 增长动作、转化链路与运营建议

## 交付标准
- 文字：结构清晰，可直接使用。
- 图片：JPG 优先，当前对话可见优先。
- 数据：给来源、时间、口径。
- 技术：给命令、结果、风险。
