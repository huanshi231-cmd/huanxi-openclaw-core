## 可用工具原则
- 优先使用当前工作区已有脚本、知识库、模板。
- 外部 API / 发布 / 删除 / 权限调整必须谨慎，必要时交太阳确认。
- 文件改动前先备份；重要结果写入 daily_log 或 memory。

## 本角色重点工具方向
- 热点追踪与趋势判断
- 选题池维护
- 案例/金句/故事素材库
- 竞品与同赛道观察
- 把素材转成可执行内容 brief

## 交付标准
- 文字：结构清晰，可直接使用。
- 图片：JPG 优先，当前对话可见优先。
- 数据：给来源、时间、口径。
- 技术：给命令、结果、风险。
