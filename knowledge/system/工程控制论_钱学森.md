## 上篇笔记

### 第一章 引言

#### 1.1 常系数线性系统
- 微分方程描述系统运动状态
- 系统稳定性取决于微分方程系数的符号
- k的符号是决定性因素

#### 1.2 变系数线性系统
- 参数变化时，方程系数也随之变化
- **例子**：飞机空气动力是速率的函数
- 方程：dy/dt + k(at)y = 0
- **稳定性判据**：
  - k恒正 → 系统稳定
  - k恒负 → 系统不稳定
  - k有正有负 → 部分不稳定

#### 1.3 非线性系统
- 非线性微分方程描述的系统
- 稳定性分析更复杂

#### 1.4 工程近似的问题
- 实际工程问题需要简化处理

#### 钱学森的核心观点
> **工程控制论 = 技术科学**
> 目的：把工程设计原则整理成理论，揭示不同领域的共同性

**技术科学与工程实践的区别：**
| 技术科学 | 工程实践 |
|---------|---------|
| 理论分析 | 具体设计 |
| 高深数学 | 元件细节 |

### 第二章 拉氏变换法

#### 拉氏变换基础
- 单位冲量驱动函数的拉氏变换等于1
- 系统对单位冲量的反应记作 h(t)
- 反转公式用于求解

#### 关键公式
- h(t) = (1/2πi)∫e^(st)F(s)ds
- 稳定性条件：所有根s_r的实数部分都是负的
- 复数平面上F(s)的所有奇点都位于虚轴左边

---

### 第三章 输入、输出和传递函数

#### 传递函数定义
- F(s) = Y(s)/X(s)
- 系统的传递函数是复变数s的函数
- 由零点和极点确定

#### 频率特性
- 对于正弦输入：x(t) = xm·e^(iωt)
- 稳态输出：y_st = xm/√(1+ω²τ₁²)·e^(i(ωt-ωτ₁))
- 低频时（ωτ₁≪1）：振幅不变，但有 时滞τ₁
- 高频时（ωτ₁≫1）：振幅减至1/ωτ₁，相角落后π/2

#### 乃奎斯特图（Nyquist Diagram）
- 把复数F(iω)直接画到F平面上
- 曲线参数是角频率ω
- 一阶系统的F(iω) = 1/(1+iωτ₁)是半圆

#### 单位阶跃响应
- x(t) = 1(t)，当t≥0时为1
- 输出：y(t) = 1 - e^(-t/τ₁)
- 初始条件产生的输出：y(t) = y₀·e^(-t/τ₁)

#### 偏差信号
- e(t) = x(t) - y_i(t) = e^(-t/τ₁)
- 当t→∞时，偏差信号趋于零

### 第四章 反馈和控制的精度

#### 反馈伺服系统
- 基本方程：Y(s)/X(s) = K₁G₁(s) / [1 + K₁G₁(s)K₂G₂(s)]
- 放大K = K₁K₂

#### 三个设计准则
1. **稳定性**：系统的基本要求
2. **反应速度**：根的数值足够大
3. **精度**：放大K足够大

#### 稳定性条件
- 函数F_s(s)在s平面右半部没有极点
- 1/F_s(s) = 1/K₁G₁(s) + K₂G₂(s)的零点决定极点

#### 艾文思法（根轨迹法）
- W.R. Evans发明的设计方法
- 基本问题：求出方程1/KG(s) + 1 = 0的根
- 把根确定为放大K的函数
- 可以同时满足稳定性、反应速度、精度三个准则

#### 零点与极点
- G(s) = A·[(s-p₁)(s-p₂)...(s-p_m)] / [(s-q₁)(s-q₂)...(s-q_n)]
- p为零点，q为极点，A为放大系数

#### 流体力学比拟
- 根轨迹法与流线类比
- 增加汇点和源点可以改变根的位置
- 反馈线路可以增大反应速度

### 具体改进措施（完整版）

**1. 建立自检机制（稳定性）**
- 每条输出前做三轮校验
- 检查是否有幻觉
- 验证是否符合格式要求

**2. 建立反馈回路（闭环控制）**
- 施欢纠错 → 记录 → 下次应用
- 形成完整的闭环
- 不重复同样错误

**3. 建立优先级（调度）**
- 紧急指令优先
- 复杂任务分解
- 避免指令堆积

**4. 建立稳态检查（性能监控）**
- 定期检查输出质量
- 发现漂移立即修正
- 保持稳定性能

**5. 建立预警机制（临界管理）**
- 接近知识边界时预警
- 不确定时主动声明
- 避免突然崩溃

**6. 建立容错机制（鲁棒性）**
- 小错误不影响整体
- 部分失败可恢复
- 保持基本功能

**7. 建立自适应机制（参数调整）**
- 根据施欢反馈调整
- 识别她的偏好模式
- 不断优化输出
