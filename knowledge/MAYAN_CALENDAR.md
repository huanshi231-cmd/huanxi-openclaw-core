- 13月亮历：自然时间观，"时间就是艺术"
- 13月亮历让人们与自然和宇宙的韵律重新合一

## 四、Kin计算方法

### 方法1：直接查表
透过13月亮历转换表，把公历生日转换为13月亮历生日（月份.日期），再查对应坐标。

### 方法2：数学计算
1. 记住基准：2012年12月25日 = Kin1 磁性红龙
2. 计算目标日期与基准日期的天数差
3. Kin数 = (1 + 天数差) mod 260，若为0则 Kin=260

### 简换算公式
```
Kin = (公历日期对应的卓尔金历序号)
```

## 六、星系印记解读

一个完整的星系印记包含5个位置：

1. **主印记（中间）**：命运图腾，你的人生主题
2. **推动图腾（下）**：隐藏潜能，挑战与机会
3. **挑战图腾（左）**：拓展力量，对面能量
4. **支持图腾（右）**：支持力量，熟悉的朋友
5. **指引图腾（上）**：更高智慧，最终活出的样子

### 合盘恒定值
- 主图腾 + 支持图腾 = 19（蓝风暴）
- 挑战图腾 + 支持图腾 = 9（红月）
- 主图腾 + 推动图腾 + 挑战图腾 + 支持图腾 = 10（白狗）

## 五个力量图腾的含义

### 1. 主印记（命运图腾）- 中间
- 生命特质和基础力量
- 如同玩游戏选择角色
- 核心天赋能量
- 有20个图腾角色可选（红龙、白风、蓝夜、黄种子、红蛇等）

### 2. 指引图腾 - 上方
- 最终活出来指引完成梦想的力量
- 是你要活出的方向

### 3. 推动图腾 - 下方（隐藏）
- 隐藏力量和激发潜能的力量
- 像大树的树根
- 很多时候自己意识不到，但他人能清晰看到
- 对于没活出自己的人，接触13月亮历可能最先需链接的就是潜能/推动力量

### 4. 挑战/拓展图腾 - 左侧
- 挑战和拓展的力量
- 需要面对和超越的课题

### 5. 支持图腾 - 右侧
- 支持你的力量
- 可以借助的资源

## 五个力量计算公式

- 主印记 = Kin对应的调性+图腾
- 指引（上方）= Kin - 1（若Kin=1则为260）
- 推动（下方）= Kin + 13（若>260则-260）
- 挑战（左侧）= Kin + 10 或 Kin - 10（取与主印记图腾相反的那个）
- 支持（右侧）= Kin - 13 或 Kin + 13（取与主印记调性相同的那个）
