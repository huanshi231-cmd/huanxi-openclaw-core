# 素材自动归档 SOP

## 触发时机

neirong 每次完成创作任务后，自动执行归档。

## 归档步骤

1. **收集本次用到的素材**
   - 所有 `temp/` 中的新增文件
   - 飞书文档中引用的图片链接

2. **LLM 自动处理**
   - 生成摘要（50字以内）
   - 打标签（来源、主题、类型）
   - 移动到对应分类目录
   - 建立反向链接（记录「哪些创作用了这个素材」）

3. **更新索引**
   - 在 `knowledge/raw/INDEX.md` 中追加本次归档记录

## 索引格式

```markdown
## YYYY-MM-DD

- [素材标题] - 来源 - 用于：[创作成品标题]
  - 标签：#水晶 #能量
```

## 健康检查（每周 cron）

1. 检测矛盾：如同一主题有两个互相矛盾的描述
2. 检测过时：如工具已改但 SOP 未更新
3. 补充缺失：通过联网搜索补充相关最新资料
4. 推荐关联：推荐新的概念关联和研究方向
