# raw 素材库

## 目录结构

```
raw/
├── articles/       # 原始文章/论文
├── images/         # 原始图片素材
├── links/          # 保存的链接及摘要
├── inspiration/    # 灵感截图/碎片
└── temp/           # 临时暂存（定期清理）
```

## 归档规则

1. 所有原始资料先进入 `temp/`
2. 由 LLM 自动分类到对应目录，并生成摘要
3. 每次创作完成后，成品自动归档到 `articles/` 或 `inspiration/`
4. 定期（每周）执行健康检查，检测矛盾和过时内容

## 命名规范

- 文件名格式：`YYYY-MM-DD_来源_标题简要.md`
- 图片：`YYYY-MM-DD_来源_描述.jpg`
