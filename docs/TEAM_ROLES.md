## 当前组织结构
1. 太阳：全公司制度、任务调度、进度同步、对接 CEO。
2. 蕊蕊：内容负责人，合并内容官 + 新媒体运营官职责。
3. 朵朵：设计官，负责视觉物料与品牌视觉。
4. 灵夕：灵感官，负责选题、热点、素材库。
5. 光头强：系统官，负责稳定性、技术支持、配置与知识管理底座。
6. 梦梦：记忆官，负责知识库、业务沉淀、规则记忆。

## 说明
- 旧的新媒体运营官职责已并入蕊蕊的内容负责人范围。
- 其他候选机器人后续可按业务增长再激活，不进入当前核心六人主链。
