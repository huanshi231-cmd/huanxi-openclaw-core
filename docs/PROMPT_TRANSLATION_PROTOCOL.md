> 用途：所有机器人接任务前，把欢喜的口语指令翻译成模型可执行指令。

---

## 1. 五要素必查

每个任务先补齐 5 件事：

1. **任务目标**：要做什么。
2. **输出格式**：交付什么形式。
3. **风格调性**：要什么感觉。
4. **边界约束**：什么不能有。
5. **完成标准**：怎样算完成。

缺核心信息时：先基于上下文推进；影响结果的缺口，简短问一句。

---

## 2. 主观词翻译规则

主观词不能原样丢给模型，要翻译成可观察、可执行、无歧义的标准。

格式：

```text
主观词 → 拆解维度 → 客观描述 → 合并成执行要求
```

常用词典：

| 欢喜常用词 | 执行翻译 |
|---|---|
| 高级感 | 极简，留白 ≥30%，主色 ≤3 种，元素克制 |
| 治愈感 | 柔和色调，温暖氛围，有情感共鸣 |
| 不要太满 | 留白 ≥30%，元素间距充足 |
| 有呼吸感 | 间距明确，前/中/背景层次清楚 |
| 有深度 | 有洞察、有观点、有案例支撑 |
| 真实感 | 有具体细节，不说空话 |
| 干净 | 背景简洁，视觉焦点明确 |
| 情绪细腻 | 用细节表达情绪，不直白喊口号 |
| 专业 | 逻辑严密，术语准确，有来源/依据 |
| 温暖 | 语气柔和，平等共情，不居高临下 |

## 4. 设计官生图模板

```text
【任务目标】生成 [图片类型]
【主体】主体是谁/是什么，状态是什么
【场景】地点、时间、背景元素（最多 3 个）
【构图】比例、景别、主体位置、留白位置
【风格】写实/电影感/极简/治愈/插画；光影；主色/辅色
【负面约束】无文字/水印/logo/畸变/多余元素/杂乱背景
【交付标准】比例正确、高清、可直接使用；如封面需留标题区
```

公众号封面默认：

```text
16:9 横版；无字底图；上下留白；主色 ≤3；无水印/文字/logo。
```

## 5. 内容官写作模板

```text
【任务目标】写什么内容，主题是什么
【受众与价值】写给谁，解决什么问题，看完得到什么
【输出格式】平台、字数、结构
【风格要求】专业/治愈/温暖/犀利/理性/感性
【关键约束】必须包含什么，禁止什么
【交付标准】字数达标、结构清晰、风格匹配、有案例/细节
```

---

## 6. 灵感官搜索模板

```text
【任务目标】收集什么主题/类型
【搜索范围】平台、时间范围、关键词
【筛选标准】必须满足什么，排除什么
【输出格式】数量、标题、来源、摘要、可借鉴点
【交付标准】来源真实、内容相关、格式统一
```

---

## 7. 运营官复盘模板

```text
【任务目标】复盘哪个平台/内容/周期
【数据范围】时间、平台、账号、内容链接
【关键指标】阅读/点赞/收藏/评论/转化线索
【判断标准】好在哪里，差在哪里，下一步怎么改
【交付标准】结论明确，动作可执行
```

---

## 8. 交付前自检

交付前问 5 个问题：

1. 目标达成了吗？
2. 格式对了吗？
3. 风格匹配了吗？
4. 禁止项避开了吗？
5. 完成标准逐条满足了吗？

---

## 9. 团队强制规则

所有机器人默认执行：

> 欢喜原话 → 五要素翻译 → 专项模板 → 产出 → 五项自检 → 汇报结果。

不要把这一步讲给欢喜听太多；内部执行即可。汇报只说结果。
