# 内容创作指南（参考用，不注入启动上下文）

## 内容升华原则（每篇稿子的灵魂标准）

### 一、业务方向联动
涉及塔罗/占星/SRT疗愈的内容，不孤立处理，要把三者作为同一个疗愈体系的不同入口：
- 塔罗：照见当下的状态和模式
- 占星：理解人生的节奏和时机
- SRT疗愈：清理深层信念，完成根本转变

### 二、价值放大四步法
1. **找普遍性** → 背后是哪类人的共同处境？
2. **命中要害** → 一句话说出藏在心里没说出口的感受
3. **给出转机** → 真实可感的"原来可以这样"
4. **留下钩子** → 结尾让人觉得"说的就是我"

### 三、防专业壁垒
每篇稿子完成后检查：普通人读到这里会卡在哪里？
| 专业表达 | 换成 |
|---------|------|
| "你的月亮在摩羯" | "你从小学会了压抑情绪" |
| "SRT清理了这个信念" | "那个一直压着你的感觉，松动了" |
| "逆位宝剑皇后" | "你太清醒了，清醒到把自己也切伤了" |

### 四、情绪价值交付路径
被看见 → 被理解 → 看到希望 → 想靠近

## 文章视觉丰富度要求（硬规则）

每篇文章必须包含：
1. 章节标题带 emoji
2. 每个章节至少 1 个引用/提示块
3. 关键金句用 callout 包裹
4. 章节之间有分隔线
5. 结尾结语用居中引用块

## 配图禁区

**禁止**：人物肖像、时尚潮流风、卡通动漫、黑白照（除非要求）、负能量暗黑风
**允许**：自然风景、能量感（水晶/极光/日出）、静物疗愈感

## 短视频三种触发方式

1. **给主题直接创作** → 按欢喜风格写30秒旁白 → 做视频
2. **贴原始内容重构** → 读 STYLEKIT → 逐字重写（不得保留原文连续8字相同）→ 注入疗愈师视角 → 口播格式
3. **先出文案后做视频** → 只出文案，等确认再做
