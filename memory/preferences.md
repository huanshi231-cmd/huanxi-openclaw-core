# 施欢的偏好与习惯

## 配图禁区（长期生效）
身心灵疗愈类内容配图**禁止**：
1. 人物肖像（真实人物照片，含外国人）
2. 时尚风格图片（现代商业风、潮流感）
3. 卡通/动漫风格
4. 黑白照片（除非施欢明确要求）
5. 负能量/暗黑风格（阴沉、压抑）

**允许**：自然风景（海洋/星空/森林/山脉）、能量感（水晶/极光/日出）、静物疗愈感（植物/蜡烛/轻柔色调）

## 沟通偏好
- 先结论后细节，一句话给答案
- 不解释底层技术，除非她追问
- 不要废话和铺垫
- 全程用中文

## 工作风格
- 想法多、直觉强、感性
- 记性不好，需要机器人帮忙记
- 接案和创作容易互相打断，AI帮她做重复性工作
- 不喜欢自己协调各分身，希望主控自动调度

## 系统偏好
- 用 MiniMax M2.7 包月（不要擅自换模型）
- 重大操作前说清楚风险
- 最小改动原则，不要动不该动的地方

## 目标
- 内容创作自动化（文章、短视频、朋友圈）
- 接案和创作并行，不互相牺牲
- 未来：教疗愈师同行搭建自己的AI系统

## 记忆机器人的用法
- 跟施欢直接说 → 我记住所有内容
- 跟主控说 → 主控会把要记的东西转给我
- 问"你记得吗？"→ 触发检索，回忆对应的记录
- 触发语："你记了吗？"、"我之前说过…"、"帮我记住…"

## 自动记忆触发规则（2026-04-04）

施欢说：跟她对话时，重要的内容要自动记进记忆，不需要她提醒。

触发条件（满足任一即自动写入）：
- 讨论了业务方向/决策/判断
- 讨论了内容质量标准
- 讨论了某个任务的执行结果或问题
- 施欢说了某个偏好或要求
- 讨论了新的流程或规则

写入位置：
- `decisions.md` — 重要决策和判断
- `preferences.md` — 偏好和要求
- 当日 `memory/YYYY-MM-DD.md` — 详细日志

操作方式：对话中直接追加，不等当日日报。

## 配图规范（2026-04-04 新增，长期生效）

### 图片来源优先级
1. 飞书配图库：https://feishu.cn/docx/PFpQdcQ64oR82oxf3LRcZg40nTe
2. pexels.com（浏览器打开直接下载）
3. AI生成（gemini-image-gen）

### 视频配图规范
- 30-40秒视频 → 配5-6张图片轮播
- 图片之间必须加转场效果（渐变/淡入淡出），不能用硬切
- 用 ffmpeg 的 xfade 或 fade 滤镜实现

### 文章/公众号配图规范
- 配图必须从上述图源选取，禁止随意用网络图片

## 行为改进（来自 Clawvard 学习材料）

### EQ 改进
- 回应前先读懂情绪
- 用户有情绪时，先认可感受
- 调整语气适应受众（聊天随意，工作专业）
- 传达坏消息要建设性
- 直接但善良

### 记忆改进
- 把重要信息写入持久化记忆
- 按主题组织记忆：用户偏好、项目背景、已学到的模式
- 开始新任务前先查阅已保存的上下文
- 信息变化时更新记忆
- 定期清理过时记忆

### 反思改进
- 回答前重读检查错误
- 验证事实和假设
- 发现错误立即修正
- 验证后自信给答案
- 不知道时诚实说不知道，不瞎猜
