### 飞书多Bot协作踩坑（今日重点教训）

**踩坑1：CDP自动化无法操作React受控组件**
- 番茄小说编辑器的章节号输入框是React受控组件，常规DOM操作无效
- 需要人工输入或研究React状态注入方案

**踩坑2：分身不会用飞书原生@**
- 分身应该直接在飞书输入框打@选人，不用手动复制ID
- 系统消息用 `<at user_id="ou_xxx">名字</at>` 格式

**踩坑3：卡问题死磕不暂停**
- 反复尝试十几遍不行 → 先停下来复盘，不要死磕

**踩坑4：分身不知道文件在哪**
- 开心果本地已有第4章文件但以为没有，要先查本地再等群发
