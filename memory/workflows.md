# 已验证可用的工作流

## 公众号一键生成
- **触发**：和内容机器人说"生成公众号文章，主题：XXX"
- **流程**：内容机器人 → 执行 PUBLIC_ACCOUNT_SKILL_CHAIN → 草稿存入公众号邮箱
- **验证**：2026-04-02 测试通过

## 朋友圈生成
- **触发**：和内容机器人说"生成今日朋友圈"
- **注意**：微信 API 调用需加 `trust_env = False`（否则代理报 appsecret missing）
- **验证**：2026-04-02 修复后通过

## 记忆记录
- **方式一**：去记忆群直接说"帮我记住：XXX"
- **方式二**：和主控说"帮我记住：XXX"，主控转发到记忆群
- **存储位置**：`memory/preferences.md`（偏好）/ `memory/decisions.md`（决策）/ `memory/YYYY-MM-DD.md`（日流水）
