# 第1章 是不是我哪里不够好

"我觉得，是不是我哪里不够好。"

林晚的手指在膝盖上蹭了蹭。右手食指的茧磨过西裤面料，发出轻微的沙沙声，像指甲刮在心上。

对面的心理医生没有说话。笔悬在本子上，墨汁在笔尖凝了一滴，没掉下来。

这是林晚第三十六次来这里。每次坐下来第一句话都是这个。医生从来没回答过，她也从来没指望得到答案。

窗外的阳光斜斜切进来，在地板上划了一道线。林晚的脚在线这边，医生的脚在线那边。像两个世界。

"上周那场婚礼，你去了。"医生终于开口。声音很平，像在说天气预报。

林晚点了点头。右手食指又蹭了一下。那个茧是握剪刀握出来的，三年了，消不下去。她是婚礼策划师，专门给别人剪幸福的形状。

"新娘在仪式开始前半小时，说要换婚纱。"医生说。

林晚的手抖了一下。

那件婚纱是她熬了三个通宵改的。从胸口到裙摆，一共两百三十七颗珍珠，她一颗一颗缝上去的。新娘说要换的时候，整个工作室的人都看着她。

她说好。

然后她一个人蹲在后台，用剪刀把那些珍珠一颗一颗拆下来。拆到第三十七颗的时候，剪刀滑了，在食指上划了一道口子。血渗出来，滴在白纱上，像一朵梅花。

她没有停。

"你当时在想什么？"医生问。

林晚的胃缩了一下。这个动作她很熟悉，每次有人问她"你在想什么"的时候，她的胃都会缩成一团。

"我在想，是不是我选的婚纱不好。"她的声音很轻，"是不是我哪里没做到位，让她不信任我。如果我够好，她应该不会临时改的。"

医生的笔终于落了下去。在本子上划了一道线。

就在这时，一个画面闪了过去。

红色的衣服。
冰凉的刀。
一个男人的背影。
还有一句话，带着血的温度："是不是我……哪里不够好？"

林晚猛地吸了一口气。

这个画面太真实了，真实到她能闻到血腥味。她的右手食指突然疼了起来，像被那把刀割了一样。

"你怎么了？"医生问。

"没什么。"林晚摇了摇头，手指握成拳，把那个疼握在掌心里，"就是有点冷。"

心理咨询室的空调开在二十六度。不冷。

医生没有戳穿她。只是看着她的右手。

"那个茧，"医生说，"你知道它是怎么来的。"

林晚的手指又蹭了一下。

"握剪刀握的。"她说。

"不是。"医生说，"在你做婚礼策划师之前，它就在了。你第一次来的时候，我就注意到了。"

林晚愣住了。

她从来没注意过这个茧是什么时候长出来的。好像从她记事起，它就在那里了。像一个胎记，像一个诅咒。

"你总是做噩梦，对不对？"医生说。

林晚的胃又缩了一下。

最近半年，她总是做同一个梦。梦里有一个穿红衣服的女人，站在一间屋子里，等着什么人。然后门开了，一个男人走进来。然后是一把刀。

每次醒来，她的右手食指都会疼。

"那个女人是谁？"医生问。

"我不知道。"林晚说，"可是我觉得……"

她停住了。后面那句话太荒谬了，她不敢说出口。

"你觉得她就是你。"医生说。

林晚的眼泪突然掉了下来。

她没有哭出声音，只是眼泪一颗一颗往下掉，砸在膝头的西裤上，晕开一小片湿痕。她的手放在腿上，手指握得很紧，指节发白。

医生没有递纸巾。就那样看着她哭。

哭了大概三分钟，林晚的眼泪停了。她用手背擦了擦脸，动作很轻，像怕弄疼自己。

"对不起。"她说。

"为什么道歉？"医生问。

"我控制不住。"林晚说，"我是不是很没用？"

医生的笔又划了一道线。

"你上个月分手了。"她说。

林晚的身体僵了一下。

那件事她没跟任何人说。连最好的朋友都不知道。她像什么都没发生过一样，每天准时上班，准时下班，准时给客户回消息。

只有她自己知道，每天晚上回家，她都会坐在地板上，坐到天亮。

"他说你太好了，好到让他有压力。"医生说。

林晚的胃缩得更紧了，疼。她弯了弯腰，用手按住肚子。

"他加班，我从来不打扰他。"她的声音有些发颤，"他说没钱，我从来不让他买单。他说不想结婚，我就从来没提过。我什么都听他的，什么都顺着他。"

她抬起头，看着医生，眼睛红红的。

"最后他说，林晚，你太好了，我配不上你。你值得更好的人。"

她笑了一下。笑起来的时候，眼角有细纹，像刀子刮过的伤口。

"你知道吗？"她说，"我当时居然还在想，是不是我真的太好了，好到让他害怕了。是不是我哪里做得不够好，才让他觉得有压力。如果我不那么懂事，不那么体贴，是不是他就不会走了。"

又是一个闪回。

烛光。
宣纸。
一个男人坐在桌前写字，她站在他身后，给他研墨。
他说："等我高中，一定来接你。"
她笑着说："好，我等你。"

林晚的手指突然疼得厉害。她把右手握在左手里，用力捏着。

"陈医生，"她的声音很轻，"是不是像我这样的人，就活该被这样对待？是不是我天生就不够好，所以无论我怎么努力，都留不住想留的人？"

医生放下笔。身体微微前倾。

"林晚，"她说，"我想请你做一件事。"

"什么？"

"现在，看着我，对我说一句话。"医生说，"不是我不够好。"

林晚看着她。嘴唇动了动。

那句话卡在喉咙里，像一块石头。她怎么也说不出来。试了三次，都失败了。

"说不出来，对不对？"医生说。

林晚的眼泪又掉了下来。这一次，她没有擦。

就在这时，手机响了。

是公司打来的。林晚接了。电话那头的声音很急，说下午那场婚礼的投资方老板要亲自过来，让她现在立刻去现场。

林晚挂了电话。站起身。腿软了一下。她扶住沙发背，站稳了。

"陈医生，我得走了。"她说。

"好。"医生也站起来，"我们下周再继续。"

林晚点了点头。拿起包，走到门口。手放在门把手上的时候，她停住了。

"陈医生，"她没有回头，"如果我永远都说不出那句话，怎么办？"

医生没有回答。

林晚打开门，走了出去。

外面的阳光很刺眼。她用手挡了挡。右手食指的茧在阳光下清晰可见，像一个烙印。

她深吸一口气。向停车场走去。

下午那场婚礼在市中心的酒店。林晚到的时候，现场已经布置得差不多了。白色的玫瑰，粉色的气球，水晶吊灯亮得晃眼睛。

她走过去，检查每一个细节。桌布有没有铺齐，花有没有蔫，音响有没有问题。检查到第三桌的时候，她听到身后有人说话。

"顾总，这边请。"

林晚的身体突然僵住了。

她没有回头。但是她知道，那个人就在她身后。隔着几米的距离，她能感觉到他的目光落在她的背上，很重，像一块石头。

她的右手食指，突然疼了起来。

像被刀割了一样。

（第1章 完）
