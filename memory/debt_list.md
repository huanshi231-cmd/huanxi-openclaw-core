# 债务清单 · 唯一真相源

> 版本：V1.0
> 日期：2026-04-16
> 维护主体：太阳（COO）

## 使用说明
- 本文件是所有任务的唯一状态源
- 所有任务必须先注册才能执行
- 任务完成后标记状态并记录交付
- 每日08:00生成债务日报

## 债务分类
- **P0**：紧急/高风险，需CEO审批
- **P1**：重要/常规，COO决策
- **P2**：日常/例行，Agent自主执行

## 任务注册格式
```
[ ] TASK-YYYY-MM-DD-序号 | 任务名称 | P0/P1/P2 | 来源 | 负责人 | 创建时间 | 截止时间 | 状态
```

## 状态说明
- **待领取**：任务已创建，未分配
- **执行中**：任务进行中
- **待确认**：执行完成，等CEO确认
- **已完成**：任务关闭
- **已取消**：任务取消
