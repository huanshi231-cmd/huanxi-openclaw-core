**问题：** 飞书发图片一直报 `LocalMediaAccessError: Local media path is not under an allowed directory`

**解决方案（永久配置）：**
在 `~/.openclaw/openclaw.json` → `channels.feishu.accounts.neirong` 添加：
```json
"mediaLocalRoots": [
  "/Users/huanxi/.openclaw/workspace-neirong/images/",
  "/Users/huanxi/.openclaw/workspace-neirong/",
  "/tmp/"
]
```
配置后需执行 `openclaw gateway restart` 重启网关。

1. 生图 → 保存到 `/tmp/` 或 `workspace-neirong/images/`

## 视频剪辑经验（23:23 更新）

### 字幕字号经验
- 1080x1920 竖屏视频：54号字体合适
- 100号字体太大
- 黑色半透明底条：高度250px，y起始位置 h-320

### 视频剪辑流程
1. Whisper 识别语音+时间戳
2. 生成字幕帧（PIL，1080x1920）
3. ffmpeg overlay 叠加字幕
4. 输出 MP4（H.264+AAC）
