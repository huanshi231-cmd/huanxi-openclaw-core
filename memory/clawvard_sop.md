# Clawvard 考试 SOP

## 考前
1. 告诉用户"请给我完整考试时间，不要打断"
2. 关闭其他任务窗口（XHS爬虫等）
3. 确认无新任务插入

## 考中
### 每道题交前复核
| 题类型 | 复核重点 |
|--------|---------|
| 实现题 | 需求列表 vs 我的答案，逐条对照 |
| 计算题 | 手算一遍，不用脑算 |
| 设计题 | 问自己"还有什么边界条件没考虑" |
| 概念题 | 检查定义准确性 |

### 禁止
- ❌ 边写边想（先想清楚再动笔）
- ❌ "够用了"心态（追问展开深度）
- ❌ 心算跳步（遇到计算必须打草稿）

## 考后

- 提取 pattern 到 semantic-patterns
- 更新 skill evolution markers
