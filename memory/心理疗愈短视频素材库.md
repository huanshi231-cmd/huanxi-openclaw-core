# 心理咨询理论与短视频内容创作素材库

## 一、主要流派理论（短视频核心素材）

### 1. 认知行为疗法 CBT
**核心观点：** 人的情绪来自想法，不是事件本身。改变想法，就能改变情绪。
**短视频角度：** "你以为的事实，其实只是你的想法"
- 举例：一个被老板批评就觉得自己不行的人，其实是把"老板批评"解释成了"我不行"
- 短视频脚本切入点：场景还原→认知扭曲揭露→换个想法

### 2. 精神分析（弗洛伊德）
**核心观点：** 现在的痛苦，根源在过去的未解决冲突。
**短视频角度：** "你为什么总是……？答案在童年"
- 原生家庭、童年创伤、未完成的情感
- 注意：不需深挖创伤，而是"意识到就疗愈"

### 3. 人本主义（罗杰斯）
**核心观点：** 每个人都本自具足，只需被无条件接纳。
**短视频角度：** "你本来就很好，只是你忘了"
- 自我接纳、自信回归、不需要外界的认可

### 4. 家庭系统排列
**核心观点：** 家庭是一个系统，个人的问题反映家庭序位问题。
**短视频角度：** "你没有错，你只是承担了不该你承担的"
- 比如：孩子承担父母争吵的压力，其实不是孩子的责任

### 5. 正念疗法
**核心观点：** 痛苦来自对过去的悔恨和对未来的焦虑，只有当下是真实的。
**短视频角度：** "此刻，你安全"
- 呼吸练习、身体扫描、接纳当下

### 6. 叙事疗法
**核心观点：** 人不是问题，问题才是问题。通过改写故事来改变人生。
**短视频角度：** "你的故事版本，不是唯一的版本"
- 比如：一个人说"我失败了"，但换个版本是"我学到了"

### 7. 萨提亚模式
**核心观点：** 每个人都有渴望：被爱、被接纳、有价值。沟通方式决定关系质量。
**短视频角度：** "你这样说话，会让关系更好"

## 二、短视频主题方向（针对疗愈+灵感生活）

### 疗愈类
1. "情绪急救"系列：愤怒/悲伤/焦虑的快速平复方法
2. "原生家庭"系列：理解父母，但不再被控制
3. "自我接纳"系列：停止自我攻击
4. "亲密关系"系列：如何健康地爱与被爱
5. "焦虑抑郁"系列：识别信号+科学应对

### 灵感生活类
1. "每日一思"系列：一个小观点，改变一天的状态
2. "内心独白"系列：疗愈系旁白+唯美画面
3. "问答"系列：你问，我答（评论区选题）

## 三、爆款短视频脚本模板

### 模板1：认知重构式（60秒）
```
[开场：一个问题场景]
你有没有过这样的经历：老板一个眼神，你就觉得自己完了？

[揭露认知扭曲]
但真相是——让你难受的，不是那个眼神，
而是你对那个眼神的解释。

[给出新视角]
如果把"他在审视我"换成"他在确认工作进度"呢？

[金句结尾]
你以为是事实，其实只是你的想法。
想法变了，情绪就变了。

[行动引导]
评论区说说你最近的一个"想法"，我来帮你拆解。
```

### 模板2：正念疗愈式（30秒）
```
[画面：自然风景，慢镜头]
[旁白：温和低沉的声线]
此刻，你在哪里？
听着我的声音，慢慢地，
把注意力放在呼吸上。
深深地吸一口气，
慢慢地，呼出来。
你没有在赶路。
这一刻，你安全。
[收尾：品牌名/话题标签]
```

### 模板3：故事疗愈式（90秒）
```
[开头：一个普遍的人生困境]
你有没有经历过这样的阶段——
什么都很好，但你就是开心不起来。

[共情]
你不是一个人。
很多人在这个阶段，会觉得自己"有问题"。

[理论引入+解答]
其实，这是因为你的内心在提醒你：
有些东西，需要被看见，被接纳。

[金句]
伤口，是光进来的地方。
你不是坏了，你只是在成长。

[结尾：话题引导]
```
