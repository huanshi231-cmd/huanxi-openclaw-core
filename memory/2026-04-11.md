
## H5微信公众号链接 - 关键经验

### 教训
- H5改版时先确认需求再动手，不要边改边确认
- 操作前先问清楚"改哪里+改成什么样"，不明确就不动
- 每次commit前截图确认效果，不要连续多次commit改同一个东西
- 遇到问题先确认是不是"原版就这样"再改回去

### GitHub Pages + 微信域名验证
- github.io域名被微信标记 → 需要去 urlsec.qq.com/complain.html 申诉
- 申诉路径：填域名 `huanshi231-cmd.github.io` → 场景选微信 → 提交 → 等1-3天
- 验证文件放仓库根目录，不是子目录
- 申诉通过后微信内可直接打开
