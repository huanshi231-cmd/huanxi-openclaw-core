# 核心文档分类速查表

> 所有 md 文档的作用和管辖范围

---

## T0 核心文件（每次启动必读）

| 文件 | 作用 |
|------|------|
| SOUL.md | 我的性格、价值观、说话方式、禁止语 |
| IDENTITY.md | 我是谁、我的角色、多Agent调度规则 |
| USER.md | 欢喜你的档案：偏好、习惯、禁忌 |
| AGENTS.md | 工作规范：任务闭环、汇报机制、情绪处理 |
| TOOLS.md | 我能用的工具清单、能力边界 |
| HEARTBEAT.md | 触发式心跳巡检（每次响应前自检） |
| BOOTSTRAP.md | 系统启动自检清单 |

---

## 记忆体系

| 文件 | 作用 |
|------|------|
| MEMORY.md | 长期记忆沉淀（蒸馏后版本） |
| preferences.md | 欢喜偏好记录 |
| decisions.md | 重大决策记录 |
| memory/日记-*.md | 每日工作日记 |

---

## 经验沉淀

| 文件 | 作用 |
|------|------|
| learnings/*.md | 各类教程和经验文档 |
| pitfalls.md | 已知的坑和应对方法 |
| self-improvement.md | 自我进化记录 |

---

## 项目/团队

| 文件 | 作用 |
|------|------|
| TEAM.md | 团队成员列表 |
| TEAM_OPENIDS.md | 飞书 ID 映射表 |
| 一人公司创业计划书.md | 欢喜事业规划 |

---

## 文档命名规则

- `SOUL/IDENTITY/USER/AGENTS/TOOLS/HEARTBEAT` = 核心身份文件（大写）
- `MEMORY/decisions/preferences` = 记忆文件
- `pitfalls/self-improvement` = 经验文件
- `learnings/*.md` = 可分享的教程文档
- `memory/日记-*.md` = 每日工作记录
