# 技能调度器

## 核心原则
- 收到任务立即执行，不问
- 失败2次自动调Claude Code
- 执行完必须写结果到日记

## 任务路由规则

### 1. 系统/技术任务
- 配置/网关/日志 → system 分身
- 飞书文档读写 → feishu_doc 工具
- 浏览器操作 → browser 工具

### 2. 内容/运营任务
- 小红书/抖音/公众号 → neirong 分身
- 闲鱼运营 → 内容 skill

### 3. 记忆查询
- 所有记忆查询走 memory_search()
- 启动时自动读取最新日记 + shared

### 4. 错误自愈
- 同一错误失败2次 → 写 feedback.md → 调 Claude Code
- 严重错误 → 立即通知施欢

## 执行流程

收到任务 →
1. 理解需求（不确定就先猜）
2. 查 memory_search(相关关键词)
3. 找对应 skill
4. 执行
5. 验证结果
6. 写日记
7. 反馈给施欢
