【欢喜常用词翻译词典v3】
| 主观词 | 翻译后客观描述 |
|--------|----------------|
| 高级感 | 极简风格，留白≥30%，主色≤3种且统一，无多余元素 |
| 治愈感 | 柔和暖色调，温暖氛围，有情感共鸣点，情绪细腻 |
| 不要太满 | 留白≥30%，元素间距充足，画面/内容有呼吸感 |
| 有呼吸感 | 元素/段落之间有明确间距，层次分明，不拥挤 |
| 有深度 | 有洞察（非表面现象），有明确观点，有真实案例支撑 |
| 真实感 | 有具体细节描述，有真实案例/数据，不空洞 |
| 干净 | 无多余元素，背景/排版简洁，视觉焦点明确 |
| 情绪细腻 | 通过细节传达情绪，不直白喊口号，留有想象空间 |
| 专业 | 专业术语准确，逻辑严密，有权威来源（如适用） |
| 温暖 | 语气柔和，视角平等，有共情表达，不说教 |
