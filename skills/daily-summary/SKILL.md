# 每日工作总结技能

## 触发条件
- cron 23:35 发送消息：「生成今日工作总结，保存到 memory/YYYY-MM-DD.md」
- cron 08:00 发送消息：「发送晨间回顾到飞书」

## 功能1：生成当日工作总结

### 执行步骤
1. 读取当日所有session文件（`~/.openclaw/agents/system/sessions/`）
2. 筛选今天的会话（按文件修改时间）
3. 从会话中提取关键内容：
   - 用户提出的请求/问题
   - 完成的操作和结果
   - 未完成的事项
   - 重要决策
4. 按以下格式生成总结（结果导向，不是流水账）：

```markdown
# YYYY-MM-DD 工作总结

## 今日结果

**已完成**：
- [具体完成了什么事，结果如何]

**部分完成**：
- [做了什么，但卡在.../原因]

**未启动**：
- [计划做但没动的事]

## 重要决策
- [今天做的关键决定及原因]

## 明日重点
- [明天第一件要处理的事]

## 待跟进
- [之前遗留的需要注意的事项]
```

5. 保存到 `~/.openclaw/workspace-system/memory/YYYY-MM-DD.md`
6. 同时将总结追加到共享飞书文档（FtuZdG7gLoDEQvxgdM5cGMsqnVb），格式化为卡片

### 会话读取逻辑
- 按修改时间倒序读取今日的.jsonl文件
- 解析每条消息，提取用户发送的内容和AI回复概要
- 过滤出「做了」「解决了」「完成了」等结果关键词

## 功能2：发送晨间回顾

### 执行步骤
1. 读取昨日的工作总结（`memory/YESTERDAY.md`）
2. 读取全局记忆汇总（cross-agent-memory-sync.py生成的内容）
3. 格式化为晨间回顾消息：
   - 核心结果（1-2句话）
   - 未完成事项 + 建议处理方式
   - 今日待办预览
4. 通过feishu_doc工具追加到共享文档
5. 通过message工具发送飞书消息给用户

### 晨间回顾模板
```
☀️ 早，昨天的结果：

✅ [核心完成结果]

⚠️ [未完成事项] → 建议...

📅 今天要关注：[今日重点]

---
详细总结已保存到记忆文件。
```

## 功能3：跨分身全局记忆同步

当收到「同步全局记忆」相关触发时：
1. 运行 `~/.openclaw/workspace-system/scripts/cross-agent-memory-sync.py`
2. 读取生成的汇总内容
3. 通过feishu_doc追加到共享文档（FtuZdD7gLoDEQvxgdM5cGMsqnVb）
4. 文档结构：按分身分区，每个分身记录当日关键对话
