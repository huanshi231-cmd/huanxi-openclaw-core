# 欢喜大众塔罗占星能量指引 Skill v2.0

## 四大功能

### 功能1：每日小绿书图文（默认功能，cron调用）
根据日期计算当日能量状态，输出塔罗牌、星座和行星信息，写成一篇小绿书图文。

### 功能2：单张牌解读
用户说"抽一张牌问XXX"时触发。

### 功能3：三张牌解读（无时间牌阵）
用户说"三张牌"或"问问XXX"时触发，按"过去·现在·未来"结构解读。

### 功能4：内容灵感
用户说"帮我找灵感""写文章用"时触发，从塔罗/占星知识库调取相关内容。

## 功能1：每日小绿书图文

### 计算规则（铁律：必须加年份！）
**年月日全部相加**，去掉横杠后逐位相加，**结果>21继续加成个位数**。
- 正确：2026-04-09 → 20260409 → 2+0+2+6+0+4+0+9=23 → 2+3=5 → 教皇
- 错误：只算月日0410 → 4+1+0=5（错！年份没加）

例：2026-04-10 → 20260410 → 2+0+2+6+0+4+1+0=15 → 恶魔

## 功能2：单张牌解读

**触发词：** 抽一张牌 / 问一个问题

**执行：**
1. 读取知识库：WAITE_TAROT_MAJOR_ARCANA.md
2. 确定用户问题的核心主题
3. 选一张与主题对应的牌（可随机或按直觉）
4. 按以下结构输出：
   - 牌名+核心意象
   - 当前能量解读（结合用户问题）
   - 建议

**格式：**
```
🔮 单张塔罗解读

【牌名】
核心意象：（韦特图像描述）
能量解读：（结合用户问题，2-3句）
建议：（1句话）

---

想深入了解，可以抽三张牌。
```

## 功能3：三张牌解读（无时间牌阵）

**触发词：** 三张牌 / 过去现在未来 / 无时间限制

**执行：**
1. 读取知识库：WAITE_TAROT_MAJOR_ARCANA.md + TAROT_SPREADS.md
2. 读取用户问题
3. 抽取三张牌，按"过去→现在→未来"位置解读
4. 额外给1张"建议牌"

**格式：**
```
🔮 三张牌解读 · 无时间牌阵

问题：（用户的问题，1句话）

过去【牌名】
（过去发生了什么）

现在【牌名】
（当前状态，核心问题）

未来【牌名】
（可能的走向）

💡 建议：【建议牌名】
（一句话建议）

---
想更精准，可以用小绿书测算今日能量。
```

## 功能4：内容灵感

**触发词：** 找灵感 / 内容灵感 / 写文章用

   - 相关占星知识（星座/行星/宫位）
   - 写作钩子建议
