# 塔罗牌阵速查（参考 tarot-spreads + tarot-reading skill）

---

## 常用牌阵一览

| 牌阵 | 牌数 | 适用场景 |
|------|------|---------|
| 单张牌 | 1 | 每日指引/快速问题 |
| 三张牌 | 3 | 简单问题/过去-现在-未来 |
| 关系牌阵 | 5 | 感情/合作关系 |
| 决策牌阵 | 3 | 二选一/该怎么做 |
| 马蹄牌阵 | 7 | 比三张深但比凯尔特十字简 |
| 凯尔特十字 | 10 | 深度解读/复杂情况 |

---

## 1. 单张牌解读

**适用：** 每日指引、快速问题、一句话答案
**位置：** 任意一张

**解读三维度：**
1. 牌的核心牌义
2. 当前能量状态（结合问题）
3. 建议/提示

---

## 2. 三张牌（无时间牌阵）

**标准布局：**

| 位置 | 含义 |
|------|------|
| 1（左） | 过去 — 导致现在状态的背景/根源 |
| 2（中） | 现在 — 核心情况 |
| 3（右） | 未来 — 如果沿着这条路走下去的走向 |

**变体：**

| 变体 | 位置1 | 位置2 | 位置3 |
|------|--------|--------|--------|
| 身心灵 | 身体/物质 | 头脑/思想 | 精神/灵魂 |
| 情况分析 | 情况 | 挑战 | 建议 |
| 决策 | 选项A结果 | 选项B结果 | 指引 |
| 关系 | 你的感受 | 对方的感受 | 关系走向 |

## 3. 凯尔特十字（10张牌，最经典深度牌阵）

**布局：**
```
      [5]
 [3] [1/2] [6]
      [4]
            [10]
            [9]
            [8]
            [7]
```

| 位置 | 名称 | 含义 |
|------|------|------|
| 1 | 现状核心 | 事情的心脏，整体基调 |
| 2 | 挑战/障碍 | 横在现状上的影响 |
| 3 | 根基 | 遥远的根源/潜意识基础 |
| 4 | 最近过去 | 最近发生了什么把你带到这 |
| 5 | 有意识的目标 | 你心中理想的结局 |
| 6 | 近期未来 | 即将展开的事 |
| 7 | 你的态度 | 你对这件事的内在立场 |
| 8 | 外部环境 | 他人/周围的影响 |
| 9 | 期望与恐惧 | 最深的渴望或隐藏的担忧 |
| 10 | 最终结局 | 如果继续这条路的结果 |

**建议解读顺序：**
1. 先读1+2（核心+挑战）建立主轴
2. 再读3+4（根基+近过去）理解怎么走到这里
3. 然后5+6（目标+近期未来）看走向
4. 最后7+8+9+10（态度+环境+期望恐惧+结局）综合

## 解读规则

### 元素互动

| 组合 | 关系 | 解读 |
|------|------|------|
| 火+风 | 支持 | 热情+思考=强劲动力 |
| 水+土 | 支持 | 情感+务实=稳定扎实 |
| 火+水 | 冲突 | 激情vs情感=动荡变化 |
| 风+土 | 冲突 | 理想vs现实=需要平衡 |
| 火+土 | 中性 | 行动需要落地 |
| 水+风 | 中性 | 感受需要理性视角 |

### 重复数字

- 2张同号：该主题被强调
- 3张同号：核心议题
- 4张同号（极罕见）：压倒性力量

### 大阿卡纳vs小阿卡纳比例

- 大多是大阿卡纳：触及生命重大转变/灵魂功课
- 大多是小学：日常情况，可掌控
- 均衡混合：日常事件中有更深层意义

### 正位vs逆位解读三角度

1. **能量受阻**：正位能量被压制无法流动
2. **能量过度**：正位特质被放大到失衡
3. **内在化**：正位表达转向内心而非外在

## 特殊牌处理

| 牌 | 常见误解 | 正确方向 |
|----|---------|---------|
| 死神XIII | 意味着死亡 | 转变和重生，一个章节的自然结束 |
| 高塔XVI | 灾难来临 | 不再服务你的东西被清除，这是解放 |
| 恶魔XV | 邪恶力量 | 认出你自己的束缚和执念，是第一步 |
| 十剑 | 最坏的牌 | 你已经触底了，拐点即将到来 |

---

## 解读合成步骤

1. **第一印象**：把整组牌作为整体感受，不要分析
2. **观察比例**：大阿卡纳vs小学，元素占比，正逆比
3. **锚定关键位置**：通常位置1和2或结局卡最重
4. **逐位解读**：按牌阵顺序，结合位置含义
5. **探索互动**：注意相邻或呼应牌之间的对话和张力
6. **编织叙事**：把所有解读织成一个连贯的故事，不是孤立列表
7. **提炼核心**：得出1-2句话承载最核心的洞见或指引

## 6. 年度脱单牌阵（6张，欢喜原创·实战验证）

**适用：** 明确时间范围的"能否遇到对的人""脱单运势"类问题
**收费建议：** 200元 / 25-30分钟

**布局（两列，左看自己·右看机会）：**
```
[1]     [4]
[2]     [5]
[3]     [6]
```

| 位置 | 牌位名称 | 对应问题 | 解读重点 |
|------|---------|---------|---------|
| 1 | 当前状态 | 你现在对感情的真实心态是怎样的？ | 案主的能量状态、开放度 |
| 2 | 卡点根源 | 为什么到现在还没遇到对的人？ | 深层模式、潜意识障碍 |
| 3 | 未来能量 | 今年接下来几个月，你的感情运势走向？ | 整体能量曲线 |
| 4 | 机会窗口 | 什么时候/什么场景最可能遇到这个人？ | 时间、场景、渠道 |
| 5 | 需要调整 | 你要放下什么/改变什么才能抓住机会？ | 可执行的具体行动 |
| 6 | 最终结果 | 12月底前，能遇到符合心意的人吗？ | 明确倾向，留有余地 |

**解读顺序：** 1→2→3→4→5→6，最后合成结论
**升单钩子：** "想深入看这个人的具体性格和相处模式，可以再加3张'对方画像'拓展"
