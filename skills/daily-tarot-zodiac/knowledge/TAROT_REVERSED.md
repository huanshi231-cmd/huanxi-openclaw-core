# 逆位牌专项解读技巧

---

## 逆位三角度解读框架（必用）

### 角度1：能量受阻
正位的能量被压制、卡住、流不动
- 例子：魔术师逆位 = 有才华但施展不出来
- 例子：战车逆位 = 有动力但方向乱，冲不出去

### 角度2：能量过度
正位的特质被放大到失衡、过度
- 例子：太阳逆位 = 太自我中心，盲目乐观
- 例子：女皇逆位 = 过度溺爱，失去边界

### 角度3：内在化
能量转向内心而非外在表达
- 例子：力量逆位 = 不是没力量，是力量用在内心挣扎上
- 例子：隐士逆位 = 不是不想独处，是躲得太深不敢出来

## 正逆位组合解读逻辑

| 组合 | 解读方向 |
|------|---------|
| 全正位 | 能量顺畅，事情按自然轨迹发展 |
| 全逆位 | 能量卡住，案主有很多内在功课要做 |
| 前正后逆 | 开始顺利，后来出现阻碍 |
| 前逆后正 | 现在困难，熬过去就顺了 |
| 中间逆位 | 核心卡点在中间那个位置 |
| 结局牌逆位 | 结果有变数，案主的行动会改变走向 |
