## 二、3步制作流程

### Step 1：告诉我主题和内容

你只需要告诉我：
- 课件主题（如"财富吸引"、"塔罗入门"）
- 每一页的文案内容
- 喜欢的颜色风格（紫/蓝/金/粉）

**示例**：
```
主题：财富吸引
风格：紫色系
第1页：标题页 - "财富吸引"
第2页：核心观点 - "你不是缺钱，你是缺频率"
...
```

### Step 2：我生成HTML文件

我会在 30秒内生成完整的 `.html` 文件，保存在你指定的位置。

### Step 3：浏览器直接打开

双击 html 文件用浏览器打开，即可全屏播放。

## 三、制作规范（我执行，你无需关心）

### 3.1 文件命名
```
h5-[主题]_[日期].html
示例：h5-财富吸引_20260419.html
```

### 3.2 页面结构
- 每页一个 `<div class="slide">`
- 第一页加 `class="slide active"`
- 每页使用不同的渐变背景

### 3.3 翻页交互
- 键盘左右方向键
- 触摸左右滑动
- 鼠标左右滑动
- 底部进度条点击

### 3.4 通用样式
```css
背景：深色渐变（#0a0a0f 底色 + 渐变色块）
字体：Noto Sans SC（Google Fonts自动加载）
标题：白色 + 强调色
正文：白色60%透明度
装饰：渐变线条、光晕效果
```

### 3.5 动画过渡
```css
transition: transform 0.5s cubic-bezier(0.4, 0, 0.2, 1);
```
