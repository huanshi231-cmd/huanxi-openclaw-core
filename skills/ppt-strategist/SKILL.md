# PPT 智能生成技能包

> 根据主题自动策划 + 生成完整 PPT

## 触发词

- "做个 PPT"
- "帮我做一个 XX 主题的 PPT"
- "生成演示文稿"
- "帮我策划 PPT"

## 技能流程

### 第一步：读取策略师提示词

读取 `prompts/ppt_master_strategist.md`，理解配色映射和布局规则。

### 第二步：分析用户需求

用户给的主题和内容，提取：
- 主题关键词
- 内容类型（数据/流程/列表/对比/时间线）
- 目标受众
- 使用场景（沙龙/课程/发布会/课程等）

### 第三步：自动匹配

根据策略师规则：
```
关键词 → 配色方案
内容类型 → 布局版式
```

### 第四步：生成 PPT

调用 Python 脚本生成 PPT：

```python
# 使用方式
from pptx_smart_generator import SmartPPTGerator

generator = SmartPPTGerator("主题关键词")
generator.add_cover("主标题", "副标题")
generator.add_content_slide("页面标题", {"内容结构"})
generator.add_ending()
generator.save("输出路径")
```

### 第五步：交付

- 生成完成后发送文件给欢喜
- 说明配色方案和页面数量

## 内容类型判断

| 内容特征 | 自动选布局 |
|---------|-----------|
| 含"步骤/如何/方法" | 流程图（带箭头） |
| 含"数据/占比/比例" | 柱状图/饼图 |
| 含"要点/清单/列表" | 卡片网格 |
| 含"对比/区别/vs" | 双栏对比 |
| 含"时间/历程" | 时间线 |
| 其他 | 卡片或列表 |

## 注意事项

1. 文字精炼，每页一个核心观点
2. 生成后发文件给欢喜，不要只说"好了"
3. 中文内容优先使用"微软雅黑"字体
4. 默认 16:9 宽屏比例
