## 能力清单

### 1. cron_cooldown_manager（冷却管理器）

**功能**：当 Cron 任务连续失败 >=2 次时，自动进入 6 小时冷却期。

**使用场景**：
- 任务一直超时失败？→ 自动冷却
- 想检查哪个任务在冷却？→ 查看冷却列表

**命令**：
- auto-check：自动检查并设置冷却
- list：查看当前在冷却的任务
- check <job_id>：检查单个任务
- set <job_id>：手动设置冷却
- clear <job_id>：清除冷却

**当前冷却中的任务**：
- 星座塔罗天象联动-07:05（连续失败2次）
- 灵感催补提醒-20:00（连续失败2次）
- 收口归档+对话回顾-23:59（连续失败2次）
- 番茄日更-每日10:00（连续失败4次）

### 2. learnings_search（经验检索）

**功能**：遇到错误时，自动检索 .learnings/ 历史经验。

---

## 冷却机制规则

| 条件 | 动作 |
|------|------|
| 连续失败 >= 2 次 | 自动进入 6 小时冷却 |
| 冷却中再次失败 | 保持冷却，不重置 |
| 修复完成 | 手动清除冷却 |
