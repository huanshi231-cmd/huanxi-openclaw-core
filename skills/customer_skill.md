# 客户服务技能

## 触发词
私信、预约、档案、回访、社群

## 执行流程

### 私信处理
1. 接收私信
2. 初步回复
3. 需求了解
4. 引导转化
5. 记录归档

### 预约管理
1. 接收预约
2. 时间确认
3. 日程协调
4. 提醒发送
5. 变更处理

### 客户档案
1. 信息录入
2. 需求记录
3. 服务记录
4. 跟进记录
5. 定期更新

### 回访反馈
1. 服务后回访
2. 满意度调查
3. 反馈收集
4. 问题跟进

## 服务SOP
1. 私信接收 → 2. 初步回复 → 3. 需求了解 → 4. 引导加微 → 5. 建立档案 → 6. 预约管理 → 7. 服务回访

## 响应标准
- 私信响应：2小时内
- 预约确认：4小时内
- 回访执行：服务后24小时
