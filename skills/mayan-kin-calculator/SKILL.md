# 玛雅历 Kin 计算器

## 功能
计算任意公历日期的玛雅历星系印记（Kin值）

## 公式
- Kin = (年份数字 + 月份数字 + 日期) % 260
- 调性 = Kin % 13（余数0→第13调性）
- 图腾 = Kin % 20（余数0→第20图腾）
